## 2. Description

TormentNexus is a local-first cognitive control plane that coordinates multi-agent LLM workflows. It's the missing layer between your AI models and the tools they need.

**What makes it different:**

- 🧠 **Progressive MCP Tool Routing** — Semantic vector search injects only the 3 most relevant tools per request. No 50K-token tool dumps.
- 🔗 **Cross-Harness Tool Parity** — Byte-for-byte identical tool signatures across Claude Code, Cursor, Codex, Gemini CLI, Copilot, and Windsurf. One config, works everywhere.
- 🌊 **LLM Waterfall** — When OpenAI rate-limits you, TormentNexus cascades to OpenRouter, then to local Ollama. Zero downtime.
- 💾 **Local-First Memory** — 14K+ persisted memories with sqlite-vec semantic search. Your team's knowledge survives restarts.
- 🤖 **Multi-Agent Swarms** — Planner, Implementer, Tester, Critic in one chatroom with consensus-driven decisions.
- 🔧 **11K+ MCP Server Catalog** — The largest indexed catalog of MCP servers with semantic auto-discovery.

**Tech stack:** Go 1.26 + TypeScript 5.x + Next.js 16 + SQLite + sqlite-vec
**License:** AGPLv3 (Community) / Commercial (HyperNexus)
**Repo:** github.com/MDMAtk/TormentNexus
