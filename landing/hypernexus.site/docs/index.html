TormentNexus

Universal AI Control Plane — Persistent memory, 20,000+ MCP tools, and multi-agent orchestration. Local-first. Open source.

⭐ GitHub 📦 npm

One Command

npx @tormentnexus/install

Detects and configures 38+ AI coding agents — Claude, Gemini, Cursor, Windsurf, OpenHands, Aider, CodeWhale, and more. Each gets MCP wiring, skills, slash commands, and memory hooks.

What It Does

🧠 Infinite Memory

Multi-tier L1/L2/L3 persistent memory. Every session searches past decisions. Store patterns, bugs, and architecture decisions.

🔧 20,000+ MCP Tools

Largest local MCP registry. Progressive semantic routing — only relevant tools enter the LLM context window.

🤖 Multi-Agent Swarm

A2A protocol with role rotation. Agents debate, bid on tasks, share context, and reach consensus autonomously.

🏢 Corporate Ready

Stripe billing, tenant Docker isolation, SSO/OIDC, RBAC, audit logging. Production deployment with live payments.

🛡️ Self-Healing

Autonomous Diagnose→Fix→Verify→Retry loop. Errors are captured, diagnosed, and remediated.

🌊 LLM Waterfall

Cascading failover: NVIDIA NIM → OpenRouter → Local Ollama. Never blocked by rate limits or outages.

Install

npx @tormentnexus/install          # One-command 38-client setup
npm install -g @tormentnexus/cli   # CLI tools
tn search "your topic"             # Search memory
tn status                          # System health
