Developer Focus

⚡

4-Layer MCP Tool Router

L1 semantic search: 384-dim all-MiniLM-L6-v2 embeddings, cosine similarity, auto-load at ≥0.85. L2 BM25 catalog ranking: tool name 10×, tags 5×, description 3×. L3 working set: 16 soft / 24 hard cap, 8 hydrated schemas max, 4 active server processes. L4 runtime proxy. 5 permanent meta-tools + 27 always-on builtins. 10-min idle LRU eviction. Zero token bloat.

🔗

MCP Gateway & Aggregator

stdio JSON-RPC 2.0 aggregator with {server}__{tool} namespacing to prevent collisions across 5 catalog sources: Glama, Smithery, MCP.run, npm, GitHub Topics. Automated API secret inference via regex for GitHub, OpenAI, Anthropic, Gemini, Slack, Stripe + 4 more. 200-event traffic inspector ring buffer. JSONL audit trail, daily rotation.

🧠

Three-Tier Memory Engine

L1: 100-entry in-RAM scratchpad (heat-score eviction). L2: SQLite vault with 384-dim vector search, FTS5 full-text, and GraphRAG edge table — 50,000 long-term capacity. L3: gzip cold archive, 90-day limbo before hard-delete. Hourly sleep cycle: ForgettingCurve decay → DreamCycle consolidation → MentalModel reflection. On session start, L2 semantic query pre-populates L1 with top-5 historical heuristics.

🔍

AST-Aware Code Indexer

TypeScript compiler API (ts.createSourceFile()) splits at exact function, class, interface, type alias, module, and enum boundaries. 1,000-char soft / 2,000-char hard chunk limits. LanceDB ANN vector search (same 384-dim model). Five always-on repograph tools: find_references, find_dependents, semantic_search, build, get. Blast-radius analysis before every change.

🌊

Resilient LLM Waterfall

Automatic cascade on every 429 or 5xx across 6 providers: Google → Anthropic → OpenAI → DeepSeek → OpenRouter → local Ollama. Provider latency and error metrics persisted to SQLite for adaptive ranking. 9,774 historical sessions already ingested and available for context retrieval.

🩹

Autonomous Self-Healing Loop

HealAndVerify pipeline: LLM diagnoses error → proposes fix → applies to isolated executor → runs tests in closed loop. Every diagnosis-fix attempt persisted to the L2 memory vault for fleet-wide learning. Stop hooks intercept session ends to audit unfulfilled promises before teardown.

SSO & RBAC

HTTP middleware validates X-Enterprise-SSO tokens via your identity provider. Three built-in RBAC roles: Admin (read/write/admin/audit), Operator (read/write/execute), Viewer (read-only). Granular resource-level authorization via SecurityProvider.Authorize(). Up to 10-node cluster support.

VPC & Air-Gapped Deployment

The entire MCP gateway, memory vault, and code indexer run fully on-premises with no data egress required. Cryptographically signed licensing. L3 cold archive in gzip-compressed JSONL stays entirely within your perimeter. JSONL audit trail in daily-rotating files ready for your SIEM.

🕸️

Multi-Agent Swarm Mesh

Four specialized roles coordinated via a type-safe A2A message broker with heartbeat monitoring and audit logging. PairOrchestrator state machine: Planner → Reviewer → Implementer → Critic. Council/Debate manager with human veto service for high-stakes decisions. 1,000-event rolling router log for full observability.
